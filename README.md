#  Library Management System

A full-stack Library Management System built using **Spring Boot**, **MySQL**, and a responsive **HTML/CSS/JavaScript** frontend. The application helps manage books and users through REST APIs, providing an efficient way to perform library operations.


##  Features

* Add, update, delete, and view books
* Manage library users
* Search books
* RESTful API architecture
*  MySQL database integration
*  Responsive web interface
*  API testing using Postman

##  Installation

### 1. Clone the repository

```bash
git clone https://github.com/CHARVIgupta14/library-management-portal.git
```

### 2. Navigate to the project

```bash
cd library-management-portal
```

### 3. Configure MySQL

Create a MySQL database and update your database credentials in the `application.properties` file.

### 4. Run the backend

```bash
mvn spring-boot:run
```

### 5. Open the frontend

Run the frontend and access the application in your browser.
